# Python Automation & Backend Ecosystem

Bienvenido a mi portafolio técnico. Este repositorio contiene 3 soluciones escalables desarrolladas en Python para resolver problemas reales de negocio: Automatización, Extracción de Datos y Desarrollo Backend.

## Proyectos Incluidos

### 1. Reporte Financiero Automatizado (ETL)
* **Stack:** Python, Pandas, ExcelWriter.
* **Problema:** Consolidación manual de múltiples fuentes de datos financieras.
* **Solución:** Script ETL que unifica archivos, limpia inconsistencias y genera reportes KPI automáticos.
* **Impacto:** Reducción de tiempo de procesamiento de horas a segundos.

### 2. Market Intelligence Scraper (RPA)
* **Stack:** Python, Selenium, BeautifulSoup.
* **Problema:** Necesidad de monitoreo de precios de competencia en tiempo real en sitios con carga dinámica (JS).
* **Solución:** Bot RPA que simula navegación humana, gestiona tiempos de espera inteligentes y extrae datos estructurados (CSV) para análisis.

### 3. StockFlow API (Backend System)
* **Stack:** Python, FastAPI, Uvicorn, Pydantic.
* **Problema:** Falta de centralización e interoperabilidad en el inventario.
* **Solución:** API RESTful robusta que actúa como fuente única de verdad.
  * Endpoints CRUD completos.
  * Validación estricta de datos (Type Safety).
  * Documentación automática (Swagger UI).
